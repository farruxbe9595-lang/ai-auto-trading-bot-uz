# AI Avto Trading Bot UZ

Bu loyiha kripto bozorini kuzatadigan, o‘zbek tilida tavsiya beradigan va sinov savdosi yuritadigan PRO arxitekturadagi botdir.

## Nima qiladi?
- Binance public API orqali narxlarni oladi
- RSI, EMA, MACD, ATR va hajmni tahlil qiladi
- Tavsiya chiqaradi: SOTIB_OLISH / SOTISH / KUTISH
- Xavf boshqaruvchisi orqali tekshiradi
- Sinov savdosi ochadi va yopadi
- Telegramga xabar yuboradi
- SQLite bazaga yozadi
- Oylik hisobot yaratadi
